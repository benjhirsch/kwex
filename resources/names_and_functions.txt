Functions
=========

High-level SpiceWrapper methods
===============================
instrument_boresight(ra_or_dec) - Calculates instrument boresight coordinates in selected inertial reference frame. Argument 'ra_or_dec' takes either 'ra' or 'dec' and determines which value is returned.
clock_angle(vector) - Projects a 3D vector from the instrument's reference frame onto the instrument's image plane, then calculates the clock angle of that vector.
quaternion_component(matrix, component) - Returns component (a, x, y, z) of specified rotation matrix.
state_vector_component(from_body, to_body, vector, component) - Calculates vector from 'from_body' to 'to_body' in selected inertial reference frame. 'vector' takes 'position' or 'velocity'. 'component' takes 'x', 'y', or 'z'.
state_vector_magnitude(from_body, to_body, vector) - Computes magnitude of vector from from_body to to_body.
instrument_position_angle(axis) - calculate position angle along axis (+x, +y, +z).
sub_point(sc_or_solar, lat_or_lon) - computes subspacecraft or subsolar point on a body with a fixed reference frame, returns either lon or lat.
sol_phase_ang(target_body, obs_body) - returns phase angle of target_body - sun - obs_body.
